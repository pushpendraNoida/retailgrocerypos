 READ ME : Retail Billing

PreRequisuite
--------------

1- JDK/JRE 1.8
2- Eclipse 3x
3- apache-maven-3.6.0
4- sonarqube 5.6.7
5- Junit 5.1.0

Execution
--------------

Import the project source in eclipse, set the compiler and jre version as 1.8 in build path.
Run the MainClass for execution. Dummy value is passed through mainclass method calling DiscountRuleEngineServiceImpl class method.
